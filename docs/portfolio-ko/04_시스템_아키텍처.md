# 시스템 아키텍처

## 전체 구조

```text
Flutter App
    |
    | REST API
    v
FastAPI Backend
    |
    | SQLAlchemy
    v
MySQL Database
```

## 프론트엔드 구조

| 모듈 | 설명 |
| --- | --- |
| `screens/` | 앱 화면 |
| `widgets/` | 공통 UI 컴포넌트 |
| `services/api_client.dart` | REST API 호출 |
| `models/` | User, RunningRecord 모델 |
| `theme/` | 다크/라이트 테마 |
| `utils/` | 포맷터 |

## 백엔드 구조

| 모듈 | 설명 |
| --- | --- |
| `main.py` | API 라우터, CORS, 보안 헤더 |
| `schemas.py` | 요청/응답 검증 |
| `models.py` | DB 모델 |
| `dependencies.py` | JWT 인증 |
| `core/security.py` | 해싱/JWT |
| `core/config.py` | 환경 설정 |
| `db.py` | DB 연결 |

## 인증 흐름

```text
로그인 요청
 -> 비밀번호 검증
 -> JWT 발급
 -> Flutter shared_preferences 저장
 -> 인증 API 호출 시 Bearer token 사용
```

## 테마 흐름

```text
설정 화면 다크 모드 스위치
 -> RunkApp state 변경
 -> MaterialApp 재빌드
 -> AppColors.configure()
 -> 전체 화면 색상 변경
```
