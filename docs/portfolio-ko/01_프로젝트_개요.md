# 프로젝트 개요

## 프로젝트명

RunK

## 한 줄 소개

러닝 기록을 저장하고 친구/피드로 공유하는 소셜 러닝 MVP

## 개발 목적

RunK는 개인 기록 중심의 러닝 앱을 소셜 경험으로 확장하기 위한 프로젝트입니다. 사용자는 러닝 기록을 저장하고, 피드에서 다른 러너의 기록을 확인하며, 친구 목록/채팅 UI를 통해 함께 달리는 느낌을 받을 수 있습니다.

## 핵심 사용자

- 혼자 러닝을 시작한 초보 러너
- 러닝 기록을 친구와 가볍게 공유하고 싶은 사용자
- 운동 기록을 앱으로 관리하고 싶은 사용자

## MVP 핵심 흐름

```text
회원가입 -> 로그인 -> 러닝 기록 저장 -> 홈/기록 확인 -> 피드 공유 -> 친구 활동 확인
```

## 주요 기능

- 이메일 기반 회원가입/로그인
- JWT 세션 복원
- 러닝 기록 저장
- 내 기록 조회
- 피드 조회
- 친구 목록/채팅 UI
- 기록 통계 화면
- 다크/라이트 테마 전환

## 기술 스택

| 구분 | 기술 |
| --- | --- |
| Frontend | Flutter |
| Backend | FastAPI |
| Database | MySQL |
| ORM | SQLAlchemy |
| Auth | JWT, bcrypt |
| Infra | Docker Compose |
