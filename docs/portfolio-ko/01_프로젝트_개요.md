# 프로젝트 개요

## 프로젝트명

런크(Runk)

## 프로젝트 유형

러닝 기록 기반 소셜 네트워크 서비스 MVP

## 개발 형태

1인 개발 프로젝트

## 개발 목적

러닝 기록을 단순히 개인 기록으로 저장하는 것을 넘어, 피드 형태로 공유하여 러너 간 동기부여를 제공하는 서비스를 구현한다.

## 핵심 사용자

- 러닝을 시작한 초보 러너
- 혼자 달리지만 기록을 공유하고 싶은 사용자
- 자신의 러닝 기록을 꾸준히 관리하고 싶은 사용자
- 추후 러닝 크루, 랭킹, 코스 추천 기능을 사용할 잠재 사용자

## MVP 핵심 목표

1단계 MVP에서는 아래 흐름이 실제로 동작하는 것을 목표로 한다.

```text
회원가입 -> 로그인 -> 러닝 기록 저장 -> 피드 조회 -> 프로필 확인
```

## 기술 스택

| 구분 | 기술 |
| --- | --- |
| 프론트엔드 | Flutter |
| 백엔드 | FastAPI |
| 데이터베이스 | MySQL |
| ORM | SQLAlchemy |
| 인증 | JWT |
| 비밀번호 보안 | bcrypt |
| 로컬 인프라 | Docker Compose |
| 개발 도구 | Android Studio, PowerShell, GitHub Desktop |

## 기능 범위

### 1단계 MVP

- 회원가입 / 로그인
- 러닝 기록 저장
- 러닝 기록 피드
- 프로필

### 2단계

- 친구 기능
- 랭킹

### 3단계

- GPS 좌표 저장
- 룰 기반 AI 피드
- 러닝 코스 추천

## 제외 범위

MVP에서는 아래 기능을 제외한다.

- 실시간 위치 공유
- 채팅
- 푸시 알림
- 결제
- 운영자 페이지
