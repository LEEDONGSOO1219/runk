# 테스트 계획 및 결과

## 테스트 범위

MVP 단계에서는 핵심 API와 Flutter 기본 화면 렌더링을 검증한다.

## 테스트 환경

| 구분 | 환경 |
| --- | --- |
| OS | Windows |
| Backend | FastAPI |
| DB | MySQL 8.0 Docker |
| Frontend | Flutter |
| API 문서 | FastAPI Swagger UI |

## 백엔드 테스트

### Smoke Test 항목

| ID | 테스트 항목 | 기대 결과 | 결과 |
| --- | --- | --- | --- |
| TC-BE-001 | `/health` 호출 | `status: ok` 반환 | 통과 |
| TC-BE-002 | 회원가입 | access token 반환 | 통과 |
| TC-BE-003 | 러닝 기록 저장 | 기록 ID 반환 | 통과 |
| TC-BE-004 | 피드 조회 | 저장된 기록 포함 | 통과 |

### 실행 명령

```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\smoke-backend.ps1
```

### 실제 확인 결과

```text
health: ok
signup: passed
create running record: passed
read feed: passed
```

## Flutter 테스트

### 정적 분석

```powershell
cd frontend
flutter analyze
```

결과:

```text
No issues found
```

### 위젯 테스트

```powershell
flutter test
```

결과:

```text
All tests passed
```
