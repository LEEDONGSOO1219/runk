# 테스트 계획 및 결과

## 테스트 범위

MVP 단계에서는 핵심 API, Flutter 정적 분석, 위젯 스모크 테스트, Windows 디버그 빌드를 검증했습니다.

## 테스트 환경

| 구분 | 환경 |
| --- | --- |
| OS | Windows |
| Backend | FastAPI |
| DB | MySQL 8.0 Docker |
| Frontend | Flutter |
| API 문서 | FastAPI Swagger UI |

```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\smoke-backend.ps1
```

## Flutter 테스트

### 정적 분석

```powershell
cd frontend
flutter analyze
```

결과:

```text
No issues found
```

### 위젯 테스트

```powershell
flutter test
```

결과:

```text
All tests passed
```

### Windows 디버그 빌드

```powershell
flutter build windows --debug
```

결과:

```text
Built build\windows\x64\runner\Debug\runk.exe
```
