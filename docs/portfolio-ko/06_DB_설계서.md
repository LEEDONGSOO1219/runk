# DB 설계서

## ERD 개요

```text
users 1 ─── N running_records
```

## users

| 컬럼 | 타입 | 제약 | 설명 |
| --- | --- | --- | --- |
| id | int | PK | 사용자 ID |
| email | varchar(255) | unique, index | 로그인 이메일 |
| username | varchar(50) | unique, index | 닉네임 |
| hashed_password | varchar(255) | not null | bcrypt 해시 |
| created_at | datetime | default now | 생성 시각 |

## running_records

| 컬럼 | 타입 | 제약 | 설명 |
| --- | --- | --- | --- |
| id | int | PK | 기록 ID |
| user_id | int | FK, index | 사용자 ID |
| distance_km | float | not null | 거리 |
| duration_seconds | int | not null | 시간 |
| pace_seconds_per_km | int | not null | 페이스 |
| run_date | date | not null | 러닝 날짜 |
| memo | varchar(255) | nullable | 메모 |
| created_at | datetime | default now | 생성 시각 |
