## 현재 진행 상태

MVP 기본 구현과 포트폴리오 문서화가 완료된 상태입니다.

- FastAPI 백엔드 API 구현
- MySQL Docker 개발 환경 구성
- Flutter 앱 화면 구조 구현
- JWT 로그인/세션 복원
- 러닝 기록 저장/조회/피드 조회
- 친구/채팅 UI
- 다크/라이트 테마 전환
- Flutter analyze/test/build 검증
- GitHub 업로드 준비
